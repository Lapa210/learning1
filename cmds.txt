git init - инициализировать репозиторий
git status - отображение актуального состояния проекта
git add *имя файла*
git commit -m "commentary"   - закоммитить изменения
git push - отправить историю изменений (коммитов) на гитхаб
git pull - подгрузить историю изменений с удалённого репозитория
git branch - посмотреть все ветки (текущая отмечена)

справка
git *команда* -help (--help - откроет в браузере)
